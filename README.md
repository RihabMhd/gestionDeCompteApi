## 🚀 Description

Cette API Laravel permet de gérer **les utilisateurs et leurs profils** via **Postman**.  
L’objectif est de démontrer la maîtrise de :

- Gestion d’identité utilisateur
- Sécurisation des routes
- Manipulation des données de profil

**Fonctionnalités :**

- Création de compte
- Connexion / obtention d’un token
- Déconnexion
- Consultation et modification du profil
- Changement de mot de passe
- Suppression de compte
- (Bonus) Authentification JWT

Toutes les routes de profil sont **protégées** et nécessitent un token valide.
